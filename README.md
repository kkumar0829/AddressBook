
# AddressBook

This project is to craete a AddressBook for the users where they can add and update their adresses. It also provide a search api to search for the adressess within a given range.

## API Reference

#### Get all items

```http
  POST register/

```
```http
  GET register/<int:id>
```
```http
  PATCH register/<int:id>
```
```http
  DELETE register/<int:id>
```
```http
  GET find/<str:latitude_begin>/<str:latitude_end>/<str:longitude_begin>/<str:longitude_end>
```
